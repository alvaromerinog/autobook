# Autobook

My ASIR final project

## Summary

This project is a secure and simply vehicle maintenances management application for mobile devices.

## Requirements

* Android 5.0+
* iOS 11.0+
* Internet connection
* 30MB free space

## Tecnologies

* Dart v2.13
* Flutter v2.2.3
* Python v3.7.11
* AWS Amplify
* AWS Cognito
* AWS API Gateway
* AWS Lambda
* AWS RDS + MariaDB
